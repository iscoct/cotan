# Instalar

En [esta página](https://go.dev/doc/install) se pueden ver diferentes maneras para instalar **go** en Windows, Linux y Mac.

## Comandos Importantes

### Hacer build

```go
go build
```

### Ejecutar tras haber hecho build

```bash
./cotan
```

### Hacer Build y Ejecutar

```go
go run .
```

### Ejecutar Tests

```go
go test
```
